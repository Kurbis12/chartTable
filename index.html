Time Value

Instruction

How to Add data:

1. Place data point on chart.
2. Place second data point and zoom out.
3. Hold cursor in middle of data line so when zoom out it will stay more in middle.

How to Edit data:

1. Placed Data points can see in data table under chart.

2. Choose data value what you want to edit and click on it.

3. In window enter your value and press OK (press cancel to cancel editing process)

4. Editing time value you need to write E.g. 22:45 (colon need to be enter by user)

5. After edit is finish data will change in chart

How to Move data:

1. To move data in chart click and hold with left mouse button data point.

2. To finish moving data point let go left mmouse button.

How to Remove data:

1. To remove data from chart click right mouse button on selected data point

2. Moving data point in normal speed so when selected data point reach other data point it will be removed automatically.

How to Save data:

1. To save data that has been placed in chart press Download Data

2. Data will download on your device in jason file
